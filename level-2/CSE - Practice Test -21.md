## Question 5
An undirected graph G has n nodes. Its adjacency matrix is given by an n × n square matrix whose (i) diagonal elements are 0‘s and (ii) non-diagonal elements are 1‘s. which one of the following is TRUE?

### Options:

A. ✅ Graph G has multiple distinct MSTs, each of cost n-1
B. ❌ Graph G has no minimum spanning tree (MST)
C. ❌ Graph G has a unique MST of cost n-1
D. ❌ Graph G has multiple spanning trees of different costs
**Correct Answer:** Graph G has multiple distinct MSTs, each of cost n-1

## Question 6
Let G be an undirected connected graph with distinct edge weight. Let emax be the edge with maximum weight and emin the edge with minimum weight. Which of the following statements is false?

### Options:

A. ✅ No minimum spanning tree contains emax
B. ❌ G has a unique minimum spanning tree
C. ❌ If emax is in a minimum spanning tree, then its removal must disconnect G
D. ❌ Every minimum spanning tree of G must contain emin
**Correct Answer:** No minimum spanning tree contains emax
